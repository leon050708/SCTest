## 测试执行说明

1. 所有测试用例使用JUnit 5框架编写
2. 测试类使用@SpringBootTest注解进行集成测试
3. 使用@Transactional注解确保测试数据不会污染数据库
4. 每个测试方法都是独立的，不依赖其他测试方法
5. 测试数据在测试方法中动态创建，测试完成后自动清理

## 测试环境要求

- Java 8+
- Spring Boot 2.7+
- H2内存数据库
- JUnit 5
- Maven或Gradle构建工具
